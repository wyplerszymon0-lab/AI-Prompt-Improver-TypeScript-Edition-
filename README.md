#  AI Prompt Improver (TypeScript Edition)

A modular AI agent built with TypeScript that transforms raw, simple user inputs into high-quality, structured prompts using industry-standard Prompt Engineering frameworks.

##  Tech Stack
- **Language:** TypeScript (Node.js)
- **Engine:** OpenAI API (Model: `gpt-4o`)
- **Runtime:** [tsx](https://tsx.is/) (Modern & Fast TypeScript Execution)

##  Project Structure
- `main.mts` - Entry point handling the CLI (Command Line Interface).
- `agent.mts` - Core logic containing the `PromptAgent` class and engineering instructions.
- `package.json` - Project metadata and dependencies.

##  Getting Started

### 1. Prerequisites
Ensure you have **Node.js** (v18 or higher) installed on your machine. You will also need an **OpenAI API Key**.

### 2. Installation
Open your terminal in the project directory and run:
```bash
npm install
npm install -g tsx
